Concise

An educational application for mobile devices which use image recognition of Vuforia and deliver information in real time. Developed in Unity using C# language.

  • Vuforia image database creation and implementation on Unity 3d.
  • Application UI implementation and make functional scripts using C# language.
  • Scanned image output video presentation on augmented reality.
  • Application closing scripts generation.
